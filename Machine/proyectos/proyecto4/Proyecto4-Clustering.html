Qu ocurre si damos a psilon un valor menor que 1? Por qu? Qu parmetros logran una buena agrupacin de los datos?

Si le damos un valor a psilon menor que 1 no agrupa bien los datos.

Para que ocurra una buena agrupacin, necesitamos poner de parmetros eps $\geq$ 2. Esto podra explicarse as:

Cuando se establece eps con un valor pequeo, se restringe el tamao del vecindario, lo que dificulta que los puntos se conecten y formen regiones densas. Como resultado, muchos puntos pueden ser etiquetados como ruido o valores atpicos en lugar de asignarse a clusters.

En DBSCAN, el valor apropiado de eps depende de la densidad y distribucin de los puntos de datos. Si eps es demasiado pequeo, es posible que los clusters no sean capaces de capturar la estructura subyacente de los datos. Por otro lado, si eps es demasiado grande, puede llevar a la fusin de clusters distintos o a la formacin de un nico cluster grande.

Con esto podemos explicar por qu no se logra identificar correctamente el grupo con un eps< 2, bsicamente por que el parametro es tan pequeo que permite que se creen clusters dentro del cluster que buscamos identificar.

def elbow_plot(data, max_clusters):
    """
    Build an elbow plot to determine the optimal number of clusters using the k-means algorithm.

    Arguments:
    - data: Input data array of shape (n_samples, n_features).
    - max_clusters: Maximum number of clusters to consider.

    Returns:
    - None (displays the elbow plot).
    """
    distortions = []
    silhouette = []
    
    for n_clusters in range(2, max_clusters+1):
        kmeans = KMeans(n_clusters=n_clusters)
        kmeans.fit(data)

        labels = kmeans.labels_
        silhouette_avg = silhouette_score(data, labels)

        silhouette.append(silhouette_avg)
        distortions.append(kmeans.inertia_)
    
    # Create subplots
    fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(10, 4))
    
    # Plotting the elbow curve
    ax1.plot(range(2, max_clusters+1), distortions, marker='o')
    ax1.set_xlabel('Number of clusters')
    ax1.set_ylabel('Distortion')
    ax1.set_title('Elbow Plot')
    
    # Plotting the silhouette curve
    ax2.plot(range(2, max_clusters+1), silhouette, marker='x')
    ax2.set_xlabel('Number of clusters')
    ax2.set_ylabel('Silhouette Score')
    ax2.set_title('Silhouette Plot')
    
    # Adjust spacing between subplots
    plt.tight_layout()
    
    # Display the plot
    plt.show()

# Example usage
elbow_plot(XPCA, max_clusters=15)

def silhouette_plot_agglomerative(data, max_clusters):
    """
    Build an silhouette plot to determine the optimal number of clusters
    using the AgglomerativeClustering algorithm.

    Arguments:
    - data: Input data array of shape (n_samples, n_features).
    - max_clusters: Maximum number of clusters to consider.

    Returns:
    - None (displays the silhouette plot).
    """
    silhouette_scores = []

    for n_clusters in range(2, max_clusters + 1):
        clustering = AgglomerativeClustering(n_clusters=n_clusters)
        labels = clustering.fit_predict(data)

        silhouette_avg = silhouette_score(data, labels)
        silhouette_scores.append(silhouette_avg)

    # Plotting the elbow curve
    plt.plot(range(2, max_clusters + 1), silhouette_scores, marker='o')
    plt.xlabel('Number of clusters')
    plt.ylabel('Silhouette Score')
    plt.title('Silhouette Plot - Agglomerative Clustering')
    plt.show()

# Example usage
silhouette_plot_agglomerative(XPCA, max_clusters=15)
